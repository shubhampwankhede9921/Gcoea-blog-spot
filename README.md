
# Gcoea blog spot

Build a full stack web app to write and publish blogs for collage student. 




## Tech Stack

**Client:** React, Bootstrap.

**Server:** Node, Express,mongoose

**Database:** MongoDB


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## Deployment

To deploy this project run

```bash
  npm start
```
